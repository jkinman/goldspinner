# goldspinner
three card poker POC using the two fish encryption
- server	node
- frontend	angular
- framework	express
- auth	passport
- encryption	https://github.com/wouldgo/twofish
- card rendering	https://tairraos.github.io/Poker.JS/
- hand evaluator	https://www.npmjs.com/package/poker-evaluator
- database	mongo
